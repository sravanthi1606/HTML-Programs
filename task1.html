Sematic Elements

A Sematic elements clearly defines its meaning to both the browser and the developer
Example : <strong>, <p>

Non-Sematic Elements

Non-Sematic Elements don't tell anything about the content they contain
Example : <div>, <span>

Block-Level Elements

A block-level elements always starts in a new line
Example : <p>

Inline Elements

An inline elements doesnot starts in a new line
Example :<a> , <i>
