Meta tag


Defination:

The tag defines metadata about an HTML document. Metadata is data (information) about data. tags always go inside the element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

Attributes

1.charset:

Specifies the character encoding for the HTML documen

2.name:

Specifies a name for the metadata

3.http-equiv:

Provides an HTTP header for the information/value of the content attribute

4.Content:

Specifies the value associated with the http-equiv or name attribute



<meta name="description">

Define a description of your webpage

Example: <meta name="description" content="Free Web tutorials for HTML and CSS">

<meta name="keywords">

Define a keywords for search engine

Example: <meta name="keywords" content="HTML, CSS, JavaScript">

<meta name="author">

Define aauthor of page

Example: <meta name="author" content="John Doe">

<meta name="viewport">

Setting the viewport to make your website look good on all devices

Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta hhtp=equiv="refresh">

Refresh the document

Example: <meta http-equiv="refresh" content="30">
