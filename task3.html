Comments

Comments are some text written in your code to give an explanation about code
syntax :< !--comment...... -->

Tags

All tags are surronded by angle brackets like < tag-name> < /tag-name>

Elements

An HTML Element is defined by a start tag ,some content and end tag.
Example: < tag-name>Content...< /tag-name>

Self closing tags

Self closing tags does not need to be closed by its closing tags which means it does not have separate closing tags.
Examples : < hr/> < br/>..etc

Block-Level elements

Block level elements always starts in a new line and the browser automatically add some white spaces before and after the element
Example : < h1>, < h2>,< p>....etc
